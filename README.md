# 🕒 Flip Clock

A modern and responsive **Flip Clock** built using **HTML, CSS, and JavaScript**. The clock displays the current **day, date, time, and AM/PM** with a smooth flip animation whenever the time changes.

## ✨ Features

* ⏰ Real-time digital clock
* 🔄 Smooth flip animation
* 📅 Displays current day and date
* 🌙 12-hour format with AM/PM
* 📱 Responsive design for desktop and mobile
* 🎨 Modern gradient UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```
Flip-Clock/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.
4. The clock will automatically display the current time.
